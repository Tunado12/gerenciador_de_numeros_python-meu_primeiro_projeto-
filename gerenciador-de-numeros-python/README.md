# Gerenciador de Números em Python

Projeto simples feito em Python para praticar:
- listas
- estruturas de repetição
- condicionais
- entrada de dados pelo terminal

## Funcionalidades
- Adicionar números
- Listar números cadastrados
- Mostrar estatísticas (quantidade, maior, menor, soma e média)

## Tecnologias
- Python (sem bibliotecas externas)

## Como executar
1. Tenha o Python instalado
2. Clone o repositório
3. Execute o arquivo:

```bash
python Gerenciador_de_Números.py
